**Raveninsta** é uma ferramenta de linha de comando (CLI) desenvolvida para facilitar investigações e consultas rápidas de perfis no Instagram. Seu principal objetivo é permitir que você encontre o ID de qualquer conta do Instagram a partir do nome de usuário, de forma semelhante à plataforma [CommentPicker](https://commentpicker.com/instagram-user-id.php). Também oferece mapeamento reverso de ID para username, com recursos extras para análise e exportação.
